Claude Code、Codex、Gemini CLI 已经成为最强的终端 AI 编程工具，但国内开发者面临两大障碍：**网络访问受限**和**国际支付不便**。API 中转站应运而生，成为连接国内开发者与海外 AI 模型的桥梁。本文从实用角度横评主流中转站，帮你做出选择。

## 一句话总结

- **官方直连**：体验最好，但需代理 + 国际信用卡，成本最高
- **编程专用中转站**：专为 Claude Code / Codex 设计，零配置上手，按量或包月
- **通用 API 聚合平台**：模型多（600+），但编程工具适配不如专用站
- **开源自建**：数据最安全，但需要技术能力和服务器

|------|----------|---------------|--------------|----------|

## 开源自建方案

适合企业用户或对数据安全有极高要求的开发者。

| 项目 | 定位 | 技术栈 | Stars |
|------|------|--------|-------|
| **OneAPI / New API** | 通用 API 网关，计费+风控+路由 | Go | 20k+ |
| **CLIProxyAPI** | 将 CLI 工具包装为 API 服务 | TypeScript | 5k+ |
| **claude-relay-service (CRS)** | Claude Code 专用镜像，支持拼车 | TypeScript | 3k+ |

自建方案的优势是**完全掌控数据流**，代码不经过任何第三方。缺点是需要自己维护服务器、处理高可用和扩容。

## 8 个核心评估维度

选择中转站时，建议从以下维度综合评估：

### 1. 倍率/单价
官方 API 价格的加价比例。主流编程中转站在 1.0x–1.5x 之间，部分平台通过充值优惠实际可低于 1.0x。

### 2. 计费模式
- **按量（Pay-as-you-go）**：适合使用量波动大的开发者
- **包月**：适合每天高频使用的重度用户
- **积分制**：灵活但需注意积分有效期

### 3. 支持的工具
确认是否支持你使用的编程工具（Claude Code / Codex / Gemini CLI / Cursor 等），以及是否提供一键配置脚本。

### 4. 支持的模型
是否覆盖你需要的模型（Sonnet 4.5 / Opus 4.6 / GPT / Gemini 等），模型更新是否及时。

### 5. 稳定性/延迟
高峰期是否限流、延迟是否稳定。建议先小额试用再决定长期充值。

### 6. 支付与发票
是否支持支付宝/微信，能否开增值税发票（企业用户刚需）。

### 7. 安全与信任
- 中转站理论上可以记录你的所有 prompt 和生成内容
- 优先选择运营时间长、有企业主体、社区口碑好的平台
- 涉及核心商业代码建议使用官方直连或自建方案

### 8. 注册门槛
是否需要VPN注册、有无免费体验额度、配置复杂度。

## 配合 CodePick 收录工具使用

### Claude Code + 中转站

```bash
# 设置中转站地址和 API Key（以 AICodeMirror 为例）
export ANTHROPIC_BASE_URL=https://api.aicodemirror.com
export ANTHROPIC_API_KEY=sk-xxxxx

# 启动 Claude Code
claude
```

### Cline / Roo Code + 中转站

在 VS Code 插件设置中，将 API Base URL 改为中转站地址，填入中转站提供的 API Key 即可。

### Aider / OpenCode + 中转站

```bash
# Aider（OpenAI 兼容格式的中转站）
export OPENAI_API_BASE=https://api.example.com/v1
export OPENAI_API_KEY=sk-xxxxx
aider --model claude-sonnet-4-5

# OpenCode（同理）
export OPENAI_API_BASE=https://api.example.com/v1
export OPENAI_API_KEY=sk-xxxxx
opencode
```

> 不同中转站的 Base URL 格式可能不同（有的用 Anthropic 原生格式，有的用 OpenAI 兼容格式），请参照各平台文档。

## 使用场景推荐

### 个人开发者，轻度使用（偶尔写代码）
→ 先用 **AnyRouter** 的免费额度体验，或选 **88Code** 的 $9.99/月入门包

### 个人开发者，重度使用（每天写代码）
→ **GAC Code 299元档** 或 **SSSAiCode 包月套餐**，性价比最高

### 团队/企业用户
→ 优先考虑能**开发票**的平台（灵芽 API、SSSAiCode），或**自建 OneAPI/New API**

### 涉及敏感代码/隐私要求高
→ **自建 CRS / OneAPI** 或直接**官方 API + 代理**

### 需要多模型切换（不止 Claude）
→ **灵芽 API**（600+ 模型）或 **OpenRouter**（300+ 模型）

## 风险提醒

1. **数据安全**：中转站可以看到你的所有请求内容（代码、prompt）。涉及商业机密的代码应使用官方直连或自建方案
2. **服务稳定性**：中小平台随时可能关停，不要一次性大额充值，建议小额多充
3. **账号风险**：部分中转站使用共享账号池，可能存在被上游封禁的风险
4. **价格波动**：中转站价格随上游 API 调价频繁变动，文中价格仅供参考
5. **合规性**：企业使用建议选择有正规主体、可开票的平台
