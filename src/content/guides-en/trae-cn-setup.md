# Trae CN Complete Getting Started Guide: Free AI IDE for China, Zero-Cost AI Coding

Trae CN is ByteDance's China-edition AI IDE built on VS Code. It is completely free, requires no VPN, and comes with built-in access to top Chinese coding models including Doubao Seed Code, DeepSeek V3/R1, Kimi K2, GLM-4.5, and Qwen3-Coder. For developers in China, this is currently the best zero-cost starting point for AI-assisted coding.
