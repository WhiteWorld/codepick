## Typeless

**一句话**：全平台 AI 智能语音输入，意图理解而非逐字转录。

Typeless 是 2026 年最受关注的 AI 语音输入工具之一。它的核心卖点不是"你说什么它打什么"，而是**理解你想表达什么**——自动去除口头禅（"嗯"、"那个"）、修正口误、格式化列表、添加标点。

### 核心功能

- **系统级输入**：在任意应用中按住快捷键说话（Gmail、Slack、VS Code、浏览器等）
- **智能编辑**：自动去除填充词、修正口误、格式化输出
- **语音命令**：选中文字后说"缩短一点"、"改成正式语气"、"翻译成英文"
- **风格学习**：根据使用习惯自适应你的表达风格
- **100+ 语言**：自动检测语言，支持中英文混合输入

### 价格

| 方案 | 价格 | 额度 |
|------|------|------|
| Free | $0 | 4,000 词/周（约 16,000 词/月） |
| Pro | $12/月（年付）/ $30/月 | 无限 |

### 平台

Mac、Windows、iOS、Android

### 优点

- 免费额度慷慨，轻度使用够用
- 全平台覆盖，手机电脑无缝切换
- 零数据留存政策，隐私保护好
- 智能编辑能力突出，输出质量高

### 不足

- 代码识别能力一般，不如 Wispr Flow
- 纯云端处理，无离线模式
- 曾被安全研究者指出隐私风险（已改进）
- 单次录音有 6 分钟上限

## Wispr Flow

**一句话**：开发者首选，代码语法识别最强的语音输入工具。

如果你主要用语音来写代码（Vibe Coding），Wispr Flow 是目前最好的选择。它的**上下文感知**能力是核心差异化——在 Gmail 里自动用正式语气，在 Slack 里切换为口语风格，在 VS Code 里识别代码语法和变量名。

### 核心功能

- **代码模式**：识别函数名、变量名，自动转换命名风格（如说"create a function called calculate total"→ `calculateTotal`）
- **上下文感知**：根据当前应用自动调整输出格式和语气
- **命令模式**：语音控制编辑操作（选择、删除、替换）
- **个人词典**：自动学习你的专业术语和代码词汇

### 价格

| 方案 | 价格 | 额度 |
|------|------|------|
| Free | $0 | 2,000 词/周 |
| Pro | $12/月（年付）/ $15/月 | 无限 |
| Enterprise | 联系销售 | 无限 + 管理后台 |

### 平台

Mac、Windows、iOS、Android

### 优点

- 代码识别准确率最高（独立测试 97.2%）
- 上下文感知自动调整语气和格式
- SOC 2 Type II + HIPAA 合规，企业可用
- 与 Claude Code、Cursor 深度集成

### 不足

- 纯云端，无离线模式
- 免费额度比 Typeless 少一半
- 价格略高于部分竞品

> 💡 **Vibe Coding 用户注意**：Wispr Flow 可与 Cursor、Windsurf、Claude Code 等 AI 编程工具配合使用，通过 Push-to-Talk 快捷键直接在 IDE 中语音输入 prompt。

## VoiceInk

**一句话**：开源、买断、纯本地——隐私极客的语音输入首选。

VoiceInk 是一个开源（GPL v3）的本地语音识别工具，基于 Whisper 模型在设备上运行，所有数据不出本机。$25 一次买断，终身免费更新。

### 核心功能

- **100% 本地处理**：基于 Whisper 模型，语音识别全部在设备上完成
- **开源透明**：GPL v3 协议，代码公开可审计
- **Power Mode**：根据当前应用自动切换预设配置
- **屏幕上下文**：感知当前屏幕内容提升识别准确度
- **99% 准确率**：本地模型的识别精度接近云端方案

### 价格

| 方案 | 价格 |
|------|------|
| Solo（1 台 Mac） | $25 一次性 |
| Personal（2 台） | $39 一次性 |
| Extended（3 台） | $49 一次性 |

> 💡 **省钱提示**：VoiceInk 开源，如果你装了 Xcode，可以直接从 [GitHub 源码](https://github.com/nicedavid98/VoiceInk) 编译使用，完全免费。

### 平台

Mac（macOS 14+，Apple Silicon）、iOS

### 优点

- 一次买断，无订阅费
- 完全离线运行，隐私最大化
- 开源可审计，信任度最高
- 价格在所有付费工具中最低

### 不足

- 仅支持 Mac + Apple Silicon
- 无 AI 智能编辑（不会自动修正口误、去填充词）
- 需要至少 8GB RAM
- 社区规模较小

## Voibe

**一句话**：为 AI 编程工具量身打造的离线语音输入。

Voibe 的独特之处在于它的**开发者模式**——专为 Cursor、Windsurf 等 AI 编程工具设计，支持文件/文件夹标签插入，可以直接用语音描述需求并附带代码上下文。100% 离线处理，延迟低于 300ms。

### 核心功能

- **开发者模式**：为 Cursor/Windsurf 优化，支持文件/文件夹 chip 插入
- **100% 离线**：所有处理在本地完成，无需网络
- **Push-to-Talk**：按住快捷键说话，松手即输入
- **自定义词典**：添加专业术语和代码关键词
- **100+ 语言**：支持多语言自动检测

### 价格

| 月付 | $4.90/月 |
| 年付 | $44.10/年 |
| 买断 | $99 终身 |

### 平台

Mac（macOS 13+）

### 优点

- 唯一提供 Cursor/Windsurf 文件标签插入功能
- 100% 离线，隐私保护最强
- 无字数限制
- 买断价合理（$99 终身）

### 不足

## 怎么选？

### 按使用场景

- **写代码为主（Vibe Coding）** → **Wispr Flow**（代码识别最强）或 **Voibe**（离线 + Cursor 集成）
- **写文档/邮件/消息为主** → **Typeless**（智能编辑最强、全平台）
- **隐私第一、不联网** → **VoiceInk**（开源买断）或 **Voibe**（纯离线）
- **零预算** → **ByeType**（免费）或 **Typeless Free**（4,000 词/周）
- **企业合规** → **Wispr Flow**（SOC 2 + HIPAA）

### 按预算

| 预算 | 推荐 |
|------|------|
| ¥0 | ByeType 或 Typeless Free |
| 一次性 ≤ ¥200 | VoiceInk（$25） |
| 一次性 ≤ ¥700 | Voibe（$99 买断） |
| 月付 ≤ ¥90 | Typeless 或 Wispr Flow（$12/月） |

### 按平台

| 平台需求 | 推荐 |
|---------|------|
| Mac + Windows + 手机 | Typeless 或 Wispr Flow |
| 仅 Mac | 五款都可，VoiceInk/Voibe 性价比最高 |
| 需要 Android | Typeless 或 Wispr Flow |

## 常见问题

**Q：语音输入真的比打字快吗？**

是的。正常说话速度约 150-180 WPM，而打字平均 40-45 WPM。加上 AI 智能编辑（自动修正、格式化），实际产出效率提升 3-4 倍。有用户报告 19 天内节省了 10 小时打字时间。

**Q：中文识别准确率如何？**

Typeless 和 Wispr Flow 的中文识别准确率在 95% 以上，支持中英文混合输入无需手动切换。VoiceInk 使用本地 Whisper 模型，中文准确率约 90-95%，取决于模型大小。

**Q：能用来写代码吗？**

可以，但体验差异大。Wispr Flow 代码识别最强，能正确转换变量名和函数名（如 camelCase）。Voibe 有专门的开发者模式。其他工具在代码场景下需要更多手动修正。

**Q：隐私安全吗？会不会泄露我的代码？**

取决于工具：VoiceInk（开源本地）和 Voibe（纯离线）最安全，数据完全不出设备。Typeless 声称零数据留存。Wispr Flow 走云端但有 SOC 2 合规认证。如果处理敏感代码，建议选择离线方案。

**Q：Typeless 免费版够用吗？**

4,000 词/周（约 16,000 词/月）对轻度使用（每天几封邮件、短消息）够用。如果用于长时间编码或写文档，很快会超额。
